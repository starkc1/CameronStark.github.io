About Me

My name is Cameron Stark; I am currently going to school at the Embry-Riddle Aeronautical University Daytona Beach Campus studying Software Engineering and a member of the Mobile Application Development Club. I have a long developed passion for coding, programming and developing that began when I was in middle school over 6 years ago. I began developing with a tool called Game Maker by Yoyo games, later acquired by Microsoft, where I developed several games that allowed me to learn design and production formatting to make a functioning and easy to use product for the mobile and desktop platforms. The games later on went into the Windows 8.1 and Windows 8.1 Phone stores. My main motivation behind the developing of these games was the monetary gain that Microsoft provided for every that is put into either or both the Windows 8.1 stores. I did this game making for several years and even spoke about it in a presentation to my Boy Scout Troop, where we were among the first to get the Game Design Merit Badge in Florida. During my Junior and Senior year of High School I began to try to make an actual application, with the Windows App Studio tool by Microsoft, that creates a mobile and desktop application with very little code knowledge, but I did learn the design process of a user interface. In total before I started to teach myself languages such as C#, Python, HTML, CSS, Bootstrap, and several others, I had 10 applications in the Windows 8.1 and Windows 8.1 phone stores, with 5 applications getting over 1,000 downloads. The first application I built with both Windows app studio and Visual Studio 2015 using C# and Xaml is the Basic Math Solver. With the knowledge, I gained from this started working on Xamarin applications for Windows 10, iOS, and Android, where I am developing the iOS and Android version of the Basic Math Solver. I also gave presentation on Xamarin.Forms at HackRiddle, which was a hackathon at Embry-Riddle Aeronautical University.
